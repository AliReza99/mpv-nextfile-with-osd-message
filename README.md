#Mpv-nextfile  
This script will force open next or previous file in the currently playing files directory.

####Settings
- filtering by filetype
- Allow/disallow looping
- linux/windows

####keybindings
- nextfile  (default SHIFT+PGDWN)
- previousfile  (default SHIFT+PGUP)
